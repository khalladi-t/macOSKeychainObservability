# macOS Keychain Observability – Event Schema

## Overview

All events generated by telemetry scripts should conform to a
consistent schema to support alerting and correlation.

This schema is conceptual and can be adapted to any SIEM platform.

---

## Core Fields

| Field | Description |
|-----|-------------|
| event_type | High-level event category |
| event_subtype | Specific detection |
| platform | Operating system |
| device_id | Unique endpoint identifier |
| user | Logged-in user |
| severity | low / medium / high / critical |
| confidence | low / medium / high |
| source | Data source |
| timestamp | Event time |

---
